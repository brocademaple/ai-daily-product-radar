## 今日趋势观察

Fresh product-shaped repos are clustering around coding-agent session control, agent-authored QA, visual feedback loops, MCP observability/brokering, and local-first personal or business operating systems built on Claude Code/Codex-style CLIs.

- Coding-agent 操作面继续细分：AgentCarto 管历史/会话，Worktrunk 管 worktree，Hover/VibePin 把浏览器与视觉反馈变成 agent 可消费的任务材料。
- MCP 从“接工具”进入“治理与可观测”：Fennec 补 browser/terminal/process runtime 视野，JanusMCP 补多身份 broker，ROIC MCP 展示垂直数据工具化。
- 本地优先仍是高频叙事，但今天更具体：localmind 把 API/embedding/mem0/RAG/MCP 装进 localhost，Dezin 把生成质量门禁放在本地 agent loop。
- 移动/IM 远程控制有中国语境信号：QQ-AI-Bridge 与 Avibe/Box 类项目共同说明 coding agent 的“离开电脑后怎么管”正在成为产品问题。
- Browser agent 方向要区分开放工具与闭源服务包装；真实 Safari/Chrome/remote Chromium 都有需求，但权限、登录态和滥用风险必须前置审计。
