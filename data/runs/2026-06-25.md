## 今日趋势观察

1. Agent-native media 正在从“生成视频 API”变成“让 agent 编写、预览、渲染可编辑视频”的工程化工作流，HyperFrames 是今天最强信号。
2. Agent 安全与治理正在前移到 egress/MCP/A2A/WebSocket 层，Pipelock、beflow、Friday、ADHDev 都把审批、证据或策略做成运行时能力。
3. 浏览器正在成为本地 agent runtime：peerd、Patchright Browser、Molio、GitNexus 说明扩展、WebGPU、WebClipper、browser-side graph/RAG 都在产品化。
4. Coding-agent CLI/工作台仍在快速分化：QodeX 偏 local-first 模型和 guardrails，ADHDev 偏 session dashboard，WispTerm/cmux 偏终端控制面，AgentWire/PocketShell 偏移动与语音输入。
5. Agent memory 进入“证据、冲突、结构化图谱”竞争期：PMB、dense-mem、Graphenium、Myco Brain、IAI PME 关注点都从简单笔记转向 provenance、graph、SQLite/Postgres/Neo4j 和 MCP 工具化。
