## 今日趋势观察

1. Agent control plane 正在分化成三层：桌面/远端协作层（Omnigent）、CLI/进程管理层（mngr）、证据门禁与 DAG 执行层（OMK）。
2. “生成可运行本地软件”比单纯生成代码更有产品信号：Ironsmith 把 prompt 直接连到 SwiftUI app bundle，是今天最强的新产品形态。
3. Voice agent 开始从 Python demo 与托管平台转向自托管 runtime：Flowcat 的 Rust 单二进制、SIP/WebRTC、无 hosted control plane 是值得重点跟踪的方向。
4. MCP 项目从“新增 connector”向“把复杂资产变成 agent 工具”移动：Huolala Figma MCP 的设计稿结构化与 asset package 输出比普通 API wrapper 更有产品价值。
5. 本地优先仍是强主题：openloops、c0、cboard、COMPASS 都强调隐私、本地状态、可审计记忆或任务管理。
