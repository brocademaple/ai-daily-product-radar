## 今日趋势观察

1. Agent skills are becoming end-to-end production surfaces: Vox Director, Marketing Studio and Generative Media Skills package media/launch work as repeatable agent workflows with scripts, examples or tests.
2. Coding-agent reliability is shifting from prompts to loops and memory: Loop.js, Mnemon and OpenLoomi all make verification, recall and workspace state explicit.
3. Local-first and self-hosted agent platforms remain active: Mnemon, OpenLoomi, DuDuClaw and Omadia emphasize ownership, auditability and deployability.
4. Agent safety tools are emerging around the local machine: Agent Install Monitor and security/log products like Sigwood respond to agents taking more real actions.
5. Search/MCP infrastructure is still noisy; candidates with source, tests, Docker or MCP package surfaces outrank README-only wrappers and awesome lists.
