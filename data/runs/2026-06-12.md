## 今日趋势观察

1. Agent 产品开始明显转向“可见工作台”：Open Design、Goose、CopilotKit、JetBrains CC GUI 都在把 agent 的工作过程从聊天框抽象成桌面、IDE、前端协议或 artifact studio。
2. 代码上下文图谱继续升温，但今天更适合区分 Graphify/Understand-Anything/CodeGraphContext：前两者偏 agent skill 和可视化理解，CGC 偏 pip 可装、MCP 可接、CLI 可验的工程工具。
3. 本地模型和执行基础设施仍在补位：Lemonade 代表“本地多模态 OpenAI-compatible server”，与 sandbox-runtime/just-bash 代表的安全执行层可以组成下一阶段 agent runtime substrate。
4. 中国市场的 AI 运维需求更可见：All API Hub、Agent Reach 都围绕账号、渠道、平台登录态、成本和连接器可靠性，是真实痛点，但也带来凭据和封号风险。
5. GitTrend 的异常高星项目仍多，今天继续把 prompt-only、guide-only、可疑源码复刻和异常星速项目放进 Skip，而不当作产品机会。
