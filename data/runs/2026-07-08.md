## 今日趋势观察

1. Agent tooling is converging on durable work state: LCO, Decapod, Kobe and Forge all treat agent work as something to index, govern, isolate or hand off.
2. Local-first agent surfaces are getting stronger: SoloMD, agterm and qmtcli keep data/control on the user machine while exposing bounded agent interfaces.
3. Coding-agent safety is moving from prompt advice to enforceable evidence: roam-code, Decapod and Wardyn all foreground audit trails, policy gates or identity.
4. Vertical agents are easiest to evaluate when they are embedded in a real channel such as Slack, WhatsApp, QMT, SEO scanners or clinical coding pipelines.
5. Pure platform visions still need runnable proof; today's stronger candidates had package manifests, tests, Docker, MCP configs, screenshots or install commands.
