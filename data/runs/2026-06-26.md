## 今日趋势观察

1. 编码 Agent 工具继续从单个 CLI 走向“多 Agent 指挥台”：Multica、Agent of Empires、Detent 都把任务分派、会话管理、验证门和 dashboard 作为产品面。
2. Agent 的上下文/权限/证据层正在独立成品类：LeanCTX、SBproxy、VERITAS OS 都在回答 agent 能看什么、能碰什么、如何留痕。
3. 终端 Agent 正在获得更多可视化和跨设备界面：Sideshow 给 agent 输出一个实时视觉面，Porterminal 把终端/MCP 控制延展到手机/网页。
4. RAG 正在被“知识编译/结构化 Wiki”替代部分场景：Synthadoc 代表从向量问答转向可读、可维护知识资产的路线。
