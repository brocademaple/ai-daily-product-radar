## 今日趋势观察

1. Agent 产品正在分化成三层：底层 agent runtime/CLI，中层 workflow backend/automation，上层 agent UI/desktop assistant。
2. IM/飞书/企微类 agent 仍是最容易落地的垂直入口，因为输入、身份、通知和审批天然存在。
3. AI workflow 平台正在靠近传统 automation 平台，MCP/agent action 会成为连接器市场的新增长点。
4. “趋势监控/知识雷达/记忆层”开始变成 agent 产品的基础能力，值得和本自动化流程互相借鉴。
