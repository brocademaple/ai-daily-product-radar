## 今日趋势观察

1. AI coding agent 生态正在从“单 agent 写代码”转向“多 agent 执行控制面”：worktree、PR、CI、日志、恢复、审批变成核心产品面。
2. MCP 的机会从 server 数量增长转向开发者工具和治理工具：inspector、eval、OAuth debugger、gateway、HITL、audit log 会逐渐标配。
3. 长时运行 agent 产品正在强调身份和触发器：email/SMS/webhook/schedule/event queue 让 agent 更像“可联系的数字同事”。
4. Agent memory/context 层出现分化：个人记忆、代码图谱上下文、session mistake mining 是不同赛道，短期更适合作为基础设施试验。
