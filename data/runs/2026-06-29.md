## 今日趋势观察

Today's product-shaped repos concentrate around agent workbenches, local agent substrates, privacy firewalls, MCP observability, and small physical/mobile surfaces that make long-running coding agents easier to operate.

- Agent 工作台继续从单一聊天窗口转向“多会话、多 worktree、多审阅”的本地控制面，`superset-sh/superset` 是今天最强活跃信号。
- 本地 agent substrate 开始变具体：`monlite` 把文档、向量、队列、cron 收进一个 SQLite 文件，`dent8` 把长期记忆加上 authority/freshness 防线。
- 隐私与安全不再只靠“不要上传”：`vault-engine` 和 `mcpsnoop` 分别把 PII 处理、MCP 调用可视化放到 agent 工作流入口。
- 垂直 copilot 值得继续找：MATLAB/Simulink Copilot 虽然 star 很少，但有真实截图、工具链和明确工程场景。
- 今日 GitHub 搜索出现大量游戏外挂/免费模型 SEO repo；这类项目不进入 Top，保留真实 Skip 证据。
