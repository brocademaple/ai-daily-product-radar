## 今日趋势观察

1. 垂直 agent 产品正在从“提示词包”升级成领域操作系统：Career-Ops、dbskill、PM Skills 都在把知识、流程、状态和产物交付放进同一个 agent 工作台。
2. Coding agent 进入模型经济学阶段：Reasonix 把 DeepSeek prefix-cache 稳定性当成产品核心，说明下一轮差异化会来自 agent loop 与模型特性的深绑定。
3. Agent UI/前端协议继续升温：CopilotKit 的 AG-UI/generative UI 方向说明“让 agent 进入真实产品界面”仍是高价值基础设施层。
4. 自托管知识工作台有产品化空间：open-notebook 这类 NotebookLM 替代品开始把 Docker、Provider 配置、本地模型和持久化作为第一屏能力。
5. Skills 生态出现两极：一边是可安装、可组合的垂直工作流，另一边是单文件/方法论集合；本雷达继续优先前者。
