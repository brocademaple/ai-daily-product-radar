## 今日趋势观察

1. 自托管 AI workspace 重新升温：Odysseus 说明“本地模型 + agent 工具 + 文档/邮件/日历 + 管理权限”正在形成大众可理解的产品形态。
2. 上下文工程正在基础设施化：Headroom 把压缩、可逆检索、agent wrapper、MCP 和失败学习放到同一个 runtime 层，说明 token 成本已经成为产品机会。
3. Skills 进入供应链阶段：Agent Skills、Graphify、last30days-skill、Taste Skill 等说明 skill 不再只是 prompt 文件，安全、安装、版本、来源和可运行性会变成竞争点。
4. Agent GUI 正在贴近具体工作台：Open Design、OpenChamber、JetBrains CC GUI 分别切入设计、OpenCode 和 JetBrains IDE，控制面越来越垂直。
5. 代码知识图谱出现第二波：Graphify / Understand Anything / CodeGraphContext 都在争夺“让 coding agent 先理解项目结构”的入口，短期需要真实代码库试跑区分效果。
